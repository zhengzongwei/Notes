# 云主机配置

## 创建用户
```shell
adduser <username>
```

## 添加权限
```shell
sudo vi /etc/sudoers

```


## 设置免密登陆
```shell
ssh-copy-id root@120.46.203.191
```

## 生成免密秘钥

```shell
# 指定长度生成rsa 密钥
ssh-keygen -t rsa -b 2048
# 或者 (根据需求选择密钥长度)
ssh-keygen -t rsa -b 4096
```

## 上传证书

```shell
ssh-copy-id root@ip
```
